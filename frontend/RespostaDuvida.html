Como resolver equações do segundo grau?

Preciso de ajuda para entender o passo a passo da fórmula de Bhaskara...

Estou estudando equações do segundo grau e preciso entender melhor como aplicar a fórmula de Bhaskara. Especificamente, tenho dificuldade em:

  • Identificar os coeficientes a, b e c
  • Entender quando usar o + ou - na fórmula
  • Interpretar quando uma equação não tem solução real

Por exemplo, na equação x² + 5x + 6 = 0, como proceder?

2 Respostas

Respondido há 1 hora Melhor resposta

Para resolver equações do segundo grau usando Bhaskara, siga estes passos:

  1. Identifique os coeficientes: em x² + 5x + 6 = 0, temos a=1, b=5 e c=6
  2. Use a fórmula: x = [-b ± √(b² - 4ac)] / 2a
  3. Substitua os valores: x = [-5 ± √(25 - 24)] / 2
  4. Simplifique: x = [-5 ± √1] / 2
  5. Encontre as duas raízes: x = -2 ou x = -3

O ± indica que você deve resolver duas vezes: uma com + e outra com -. Se b² - 4ac < 0, não há raízes reais.

Respondido há 30 minutos

Você só precisa multiplicar todos os números e dividir pelo primeiro número da equação. Por exemplo:

Em x² + 5x + 6 = 0, multiplique 5 × 6 = 30, depois divida por 1.

A resposta é x = 30.
