Como funciona o ColaBora?

O ColaBora é uma comunidade de aprendizado colaborativo onde estudantes e professores se unem para compartilhar conhecimento. Nossa plataforma permite que você faça perguntas sobre qualquer disciplina e receba respostas detalhadas de outros membros da comunidade. Com explicações passo a passo e uma abordagem didática, tornamos o aprendizado mais acessível e eficiente.

Questão de Matemática:

Como resolver a equação quadrática: x² - 5x + 6 = 0?

Resposta:

Para resolver esta equação quadrática, vamos usar a fórmula de Bhaskara:

x = (-b ± √(b² - 4ac)) / 2a

Onde:

  • a = 1
  • b = -5
  • c = 6

Substituindo na fórmula:

x = (5 ± √(25 - 24)) / 2

x = (5 ± √1) / 2

x = (5 ± 1) / 2

Portanto, as soluções são:

x₁ = 3 e x₂ = 2
